# CodeAlpha_ChatbotFAQ

## Task 2: FAQ Chatbot using NLP

This project is a simple rule-based chatbot that answers Frequently Asked Questions (FAQs) using Natural Language Processing techniques.

### Features
- Preprocesses user input using NLTK (tokenization, lowercasing, punctuation removal)
- Uses TF-IDF Vectorization to convert text into numerical features
- Matches user questions with the most relevant FAQ using Cosine Similarity
- Returns the best matching answer from a predefined FAQ dataset

### Technologies Used
- Python
- NLTK
- Scikit-learn (TF-IDF, Cosine Similarity)
